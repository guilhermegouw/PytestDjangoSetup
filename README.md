# PytestDjangoSetup

Repositório criado para ser usado como modelo de configuração do Pytest no Django.

1. Criar ambiente virtual
2. Instalar o Django
3. Instalar o pytest-django
4. Criar o arquivo "pytest.ini"

Cobertura de testes
